Hashing with no Salt

Hashing a password with no salt. This means that there are no characters added to a password prior to hashing to obfuscate the actual password.

HOW TO USE

1.Either enter a password into the indicated field or press the random button to generate a random password (there might be a delay).

2. Press the HASH button to hash the password.

3. Observe the hashed password.

Hashing with Salt

Hashing a password with salt. This means that there are characters added to a password prior to hashing to obfuscate the actual password.

HOW TO USE

1. Either enter a password into the indicated field or press the random button to generate a random password (there might be a delay).

2. Enter a salt for the password or press the random salt button to generate a random salt to add to the password. For simplicity, the salt will be added after the password, but salt could be added before or after.

3. Press the HASH button to hash the password.

3. Observe the hashed password.
